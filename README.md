# 18_19_Season_Competetion_Code
 Backup Copy

该库为2018/19赛季机甲大师赛-国际预选赛中，西交利物浦大学与利物浦大学联队于比赛结束后整理的参赛代码，用于日后参考。该套代码主要包含五部分内容：

1. RM Standard Robot Open Source Code.rar

	* DJI提供的官方步兵工程文件

2. XJTLU_Standard_Infantry

	* 由 杨子琦 完成的新旧两版步兵工程文件

	2.1. standard_Infantry_new

	* 特点为裁判系统的参数读取与使用

	2.2. standard_Infantry_old		

3. XJTLU_Hero_Robot

	* 由 张申健 完成的英雄机器人工程文件

4. XJTLU_Standard_Engineer

	* 由 郭子赫 完成的工程机器人工程文件

5. XJTLU_Infantry_Master

	* 由 石家丞 完成的两版步兵工程文件（异于官方步兵的代码逻辑）

	5.1. Infantry No.3

	* 特点为小陀螺功能

	5.2. Infantry No.5

	* 特点为扭腰功能（官方代码亦有）
